inlinestatic

Decide if the vector v1 is close to the vector v2 (component by component) up to the absolute error and the relative error.

It uses a different tolerance (given in the vector v_tol) for each comparison.

If the size of the vectors are differents, it only perform the comparison between the communs components (up to the minimal size of the vectors).
